## [Install](#contents)

Pick your poison...

With the RubyGems CLI package manager:

`$ gem install raketary`

Manually:

```
$ git clone 'https://github.com/esotericpig/raketary.git'
$ cd raketary
$ bundle install
$ bundle exec rake install:local
```
